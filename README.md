# Directory layout
```bash
.
├───hdl                        # HDL code
├───inc                        # Files which declare all constants and parameters
├───libs                       # Sub-modules used in this design
│   ├───fixed_point_arithmetic # Fixed-point Arithmetic IPs
│   └───processing_element     # Processing Element
└───sim                        # Simulation files
    ├───dataset                # Dataset
    ├───testbench              # Top testbench
    └───work                   # Scripts used for simulation phase
```
